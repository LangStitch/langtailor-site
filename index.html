Install notes

  • Windows: Run the setup.exe installer and launch LangTailor from the Start menu.
  • macOS: Open the DMG and drag LangTailor to Applications. If Gatekeeper blocks an unsigned DMG, Right-click → Open, then confirm Open.
  • VS Code / VSCodium: Install from Open VSX, or run codium --install-extension langstitch.langtailor-canvas (swap codium for code in VS Code).

Full control

What we collect (with consent)
